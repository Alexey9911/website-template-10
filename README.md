## ğŸ¨ Customization

### Colors
Edit the CSS custom properties in `css/globals.css`:
```css
:root {
  --bg: #f3f3f0;
  --bg-dark: #191b20;
  --text: #000;
  --text-light: #f3f4ee;
  --accent: #ab9cea;

```

- Archive items
- Contact details
